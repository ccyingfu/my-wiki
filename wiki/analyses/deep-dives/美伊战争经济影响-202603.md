# 美伊战争全球经济影响深度分析

## 资料概要

综合3份核心报告，覆盖冲突后3/6/12个月三种情景分析：
- 美伊战争世界经济影响分析报告
- 美伊战争走向与全球影响深度研究报告
- 油价突破200美元的全球冲击分析

## 核心分析

### 三种情景核心数据

| 指标 | 基准情景（3月内缓和） | 中性情景（6个月） | 悲观情景（12个月+） |
|------|---------------------|-----------------|-------------------|
| 全球GDP增速 | 2.5-3.0% | 1.8-2.5% | 0.5-1.5% |
| 全球CPI | +0.3-0.5pp | +1.0-2.0pp | +2.5-4.0pp |
| 布伦特油价 | $100-120 | $130-170 | $200+ |
| 全球贸易增速 | -0.5pp | -1.5-2.5pp | -3.0-5.0pp |

### 各国/地区影响对比

| 地区 | 3个月GDP影响 | 脆弱性评估 | 核心风险 |
|------|-------------|-----------|---------|
| **中国** | 4.5-5.0% | 韧性最强 | 能源进口成本上升，但内需+供应链替代提供缓冲 |
| **美国** | 1.5-2.0% | 中等 | 页岩油增产对冲部分冲击，但通胀压力显著 |
| **欧盟** | 0.5-1.0% | 最脆弱 | 能源依赖进口，工业链受损严重 |
| **日本** | 0.5-1.0% | 高脆弱 | 几乎全部原油进口，日元贬值加剧输入性通胀 |
| **韩国** | 1.0-1.5% | 中高 | 能源进口+半导体出口链双重打击 |
| **东南亚** | 3.0-4.0% | 中等 | 部分产油国受益，制造国受成本冲击 |

### 四条传导链条

```
能源通道封锁（霍尔木兹）
  → 油价飙升（$100→$200+）
    → 全球通胀上升（CPI +0.5~4pp）
      → 央行被迫收紧货币政策
        → 经济增长放缓 / 滞胀风险
```

1. **能源通道**：霍尔木兹海峡封锁，全球20%原油供应中断
2. **通胀传导**：油价→交通→工业品→食品→核心CPI
3. **政策收紧**：央行在增长与通胀间两难，大概率维持偏紧
4. **经济放缓**：高利率+高通胀+低信心，消费与投资双双承压

## 关键数据

- 霍尔木兹海峡每日运输量：约2,000万桶原油
- 全球SPR（战略石油储备）总量：约15亿桶，可覆盖约75天
- OPEC+剩余产能：约300-400万桶/日（沙特为主）
- 美国页岩油增产周期：6-12个月，短期弹性有限

## 核心判断

1. **中国韧性最强**：内需市场庞大、产业链完整、能源多元化（俄罗斯管道油+中亚管道气），3个月GDP仍可维持4.5-5.0%
2. **欧洲最脆弱**：能源进口依赖度高、工业基础直接受损、地缘风险溢价最大
3. **滞胀风险上升**：若冲突持续6个月以上，全球经济面临"低增长+高通胀"的滞胀格局
4. **大宗商品分化**：能源>有色金属>农产品>贵金属的受益排序

## 投资启示

- **超配**：黄金（避险+通胀对冲）、中国资产（相对韧性）、军工
- **标配**：有色金属（铝/铜供需支撑）、美国页岩油
- **低配**：欧洲资产、日本资产、高估值成长股
- **对冲**：做多波动率（VIX）、做空欧元
