## 三、四大传导机制

```
霍尔木兹封锁 → 油价暴涨(+30~76%) → 全球通胀飙升
     ↓
航运中断(绕行好望角+40%) → 供应链受阻 → 成本上升
     ↓
风险偏好恶化 → 资本避险 → 新兴市场资本外流
     ↓
央行政策收紧(降息暂停/加息) → 经济增长放缓 → 滞胀风险
```

## 五、对中国的详细影响

### ✅ 缓冲因素
1. **战略石油储备**：约90天净进口量覆盖
2. **能源结构**：煤炭占一次能源消费56%+
3. **替代供应**：俄罗斯管道原油（占进口近20%）
4. **政策工具箱**：财政+货币双轮驱动能力

### ⚠️ 风险敞口
- 37.7%的进口石油经霍尔木兹海峡
- 中东LNG供应约占中国进口25-30%
- "一带一路"中东段投资项目安全受威胁
- 出口受全球需求放缓拖累

### 📊 预测
- CPI：3个月升至1.5-2.0%，1年达2.5-3.5%
- GDP：3个月维持4.8-5.0%，1年降至3.0-3.8%（仍全球最快）
- 人民币：有贬值压力但资本管制+3万亿外储可防危机

## 七、战后恢复：不会"一键重启"

即使海峡重开：
1. **油价黏性**：补充战略储备需求支撑高位震荡
2. **物流成本**：保险费暴涨50%+，航程成本永久性上移
3. **芯片产业链**：大厂囤货挤压中小企业，供应链重塑
4. **美元体系**：石油人民币加速兴起，美元霸权裂痕扩大
5. **地缘格局**：多极化加速，中东寻求外交多元化
