# 铝产业链专题分析

## 资料概要

基于铝产业链专题分析报告，系统梳理从上游原材料到下游终端需求的完整产业链图谱，重点分析AI时代铝需求的结构性增长机遇。

## 核心分析

### 价格走势

- LME铝价从2025年底$2,400/吨飙升至2026年4月初$3,450-3,520/吨，涨幅超40%
- SHFE铝价同步上涨至21,000-22,000元/吨
- 创2024年4月以来新高，处于2022年历史高位附近

### 产业链全景

```
上游：铝土矿 → 氧化铝 → 电解铝（原铝）
中游：铝加工（板/带/箔/型材/锻件）
下游：建筑/交通/包装/电力/电子/军工
```

### 成本结构

| 成本项 | 占比 | 说明 |
|--------|------|------|
| 电力 | 35-40% | 电解铝耗电约13,500度/吨，能源成本核心 |
| 氧化铝 | 25-28% | 铝土矿→氧化铝→电解铝 |
| 预焙阳极 | 10-15% | 石油焦+煤沥青 |
| 人工及其他 | 15-20% | 包含运输、环保等 |

### 全球产能格局

- **全球总产能**：约7,900万吨/年
- **中国产能**：约4,500万吨（占57%），受4,500万吨产能天花板约束
- **海外产能**：约3,400万吨，主要集中在俄罗斯、印度、加拿大、挪威、中东
- **中东减产**：卡塔尔停产60万吨，伊朗减产30万吨（冲突+制裁双重影响）

## 六大产业链需求分析

### 1. AI数据中心
- 机架散热系统、线缆、机壳大量用铝
- 单个大型数据中心铝用量达数万吨
- 2026年AI投资热潮驱动需求快速增长

### 2. 光伏
- 光伏边框、支架、接线盒铝件
- 全球光伏装机量年增20%+，铝需求同步增长

### 3. 储能
- 电池壳体、结构件、散热板
- 储能装机量翻倍增长，铝材需求弹性大

### 4. 电网改造
- 输电线路（铝导体替代铜）、变压器外壳
- 全球电网老化+新能源并网需求驱动投资

### 5. 军工
- 装甲、航空铝材、弹体壳体
- 地缘冲突升级推动军工订单增长

### 6. 新能源车
- 车身轻量化（铝板、挤压型材）
- 单车用铝量从120kg向200kg+提升
