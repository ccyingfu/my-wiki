# 地缘政治

## 一句话定位
地缘政治是影响全球资本市场和中国科技产业链的核心变量，中美博弈、出口管制、关税政策等持续重塑产业格局。

## 核心议题

| 议题 | 关键内容 | 影响板块 |
|------|---------|---------|
| 半导体出口管制 | 美国对华AI芯片/设备/EDA禁令 | 半导体、AI算力 |
| 关税壁垒 | 新能源车/光伏/钢铁等关税 | 新能源车、光伏、基建 |
| 生物安全法案 | CXO供应链限制 | 医疗、药明康德 |
| 数据安全 | 跨境数据流动监管 | 信息技术、金融科技 |
| 能源安全 | 核能/油气供应链 | 能源、核电 |

## 对产业链的影响

```
美国出口管制 → 半导体国产替代加速 → 设备/材料板块受益，但先进制程受限
关税壁垒 → 中国企业加速出海/本地化生产 → 龙头全球化能力分化
```

## 风险等级

| 风险等级 | 议题 | 近期动态 |
|---------|------|---------|
| 🔴 高风险 | 半导体出口管制 | 持续升级，AI芯片限制加码 |
| 🟡 中风险 | 关税政策 | 新能源车关税博弈中 |
| 🟡 中风险 | 生物安全法案 | CXO板块持续承压 |
| 🟢 低风险 | 能源安全 | 核电审批加速，影响正面 |

## 近期催化 / 风险

- **催化**：
  - 管制放松信号(外交谈判突破)
  - 国产替代突破性进展
  - 中国企业海外建厂成功(绕过关税)
- **风险**：
  - 半导体管制进一步升级至成熟制程
  - 新能源车关税全面加征
  - 中美关系恶化超预期
