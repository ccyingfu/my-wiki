# 核心-卫星配置策略（宽基ETF）

## 核心思想
以宽基ETF为工具的长期投资策略。核心仓位(70%)配置大中盘宽基指数获取市场平均收益，卫星仓位(30%)配置成长型和防御型ETF增强收益和降低波动。通过纪律化定投+再平衡实现15年资产增值。

## 要点拆解

### 关键参数

| 参数 | 设定值 |
|-----|-------|
| 初始投入 | ¥40,000 |
| 年度追加 | ¥12,000（每年1月） |
| 投资期限 | 15年（2026-2041） |
| 目标年化 | 10% |
| 总投入本金 | ¥220,000 |
| 目标终值 | 约¥737,000 |

### 再平衡规则

- 检查频率：每半年（6月/12月）
- 触发阈值：任一ETF偏离目标比例>=5个百分点
- 极端触发：单ETF偏离>=10个百分点立即执行
- 预期收益提升：再平衡额外贡献约0.5-1%年化

## 局限性

- 10%年化为长期历史均值假设，实际可能偏离
- 科创50波动率大(30-35%)，15%配置仍可能带来显著回撤
- A500ETF规模较小(25.4亿)，存在折溢价风险
- 15年投资期内必然经历2-3次30%+回撤
