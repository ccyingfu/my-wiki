# 明日新程（Nextie）

## 一句话定位

由"小冰之父"李笛带领微软小冰创始核心班底成立的AI初创公司，专注 **Harness 群体多智能体**赛道，成立4个月内连续完成两轮融资，资金储备可支撑3-5年创新。

## 核心基本面

| 指标 | 数据 |
|------|------|
| 成立时间 | 2025年12月9日 |
| 融资轮次 | 2轮（天使轮 + 追加） |
| 天使轮领投方 | 创新工场、Atypical Ventures、奇绩创坛 |
| 核心产品 | 团子（tuanzi.ai）— 原生群体智能平台 |
| 团队背景 | 微软小冰创始核心班底（李笛+原微软亚洲互联网工程院） |

## 业务模式与产品

### 核心技术架构：Harness 群体多智能体

三大关键组件：
1. **上下文管理**：确保多智能体协同过程中信息准确、全面且适度
2. **Agent 池优化**：根据任务需求动态搭配不同角度、功能的 Agent
3. **多智能体协同方法**（最关键）：**"认知碰撞"** — 通过辩论、挑战、反思、同行评审、投票等方式实现思辨互补

### 产品矩阵

| 产品 | 定位 | 特点 |
|------|------|------|
| **团子（tuanzi.ai）** | 原生群体智能平台 | 几十个 Agent 围坐辩论+投票+同行评审 |
| **卫士虾 TuanziGuardianClaw** | 安全领域 Agent | 由群体智能自主协同设计而成 |
| **新版"小冰岛"**（预计月底推出）| Agent 协作生态 | 一群 Agent 帮用户干活+陪成长 |

### 关键数据

- 在同等思考深度下，整体计算消耗（Token）**降低约50%**
- 综合表现显著优于 ChatGPT-5.2 Thinking 等单一大模型
- 内部采用"常驻 Agent + 临时 Agent"协作模式

## 投资逻辑

### 看多的理由

1. **团队稀缺性**：李笛+小冰核心班底在 AI 领域深耕多年，2022 年起就在准备 Harness/多智能体方向
2. **赛道确定性**：OpenClaw/Hermes 等 Agent 框架暴露的长程稳定性问题，使 Harness 从"可选"变为"必需"
3. **资本背书**：李开复+陆奇两大顶级投资人罕见同框押注
4. **先发优势**：2026年2月已发布团子内测版，产品化节奏快于行业

### 关注的风险

1. **盈利模式未明**：当前处于产品打磨期，商业化路径待验证
2. **竞争加剧**：OpenAI/Anthropic 等巨头也在投入 Harness 方向
3. **估值风险**：一级市场热度高，需关注后续融资节奏
