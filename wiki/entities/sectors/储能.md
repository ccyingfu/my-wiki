# 储能产业链

## 一句话定位
储能是新型电力系统和AI数据中心的刚需基础设施，大储+工商储+户储三赛道并行，AI数据中心配储成为新增量。

## 产业链结构

```
上游：电芯/PCS/BMS → 中游：储能系统集成 → 下游：发电侧/电网侧/用户侧
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-电芯 | 磷酸铁锂/钠离子电池 | 成本持续下降，产能扩张 |
| 上游-PCS | 储能逆变器 | 量价齐升 |
| 上游-BMS | 电池管理系统 | 技术升级中 |
| 中游-集成 | 储能系统集成 | 市场竞争加剧，龙头优势明显 |
| 下游-大储 | 电网侧/电源侧 | 强制配储政策驱动 |
| 下游-工商储 | 工商业用户 | 经济性拐点接近 |
| 下游-户储 | 家庭储能 | 海外市场为主 |

## 近期催化 / 风险

- **催化**：
  - AI数据中心配储需求爆发，成为新增长极
  - 电力市场化改革推进，储能盈利模式改善
  - 工商储经济性拐点临近，装机量爆发
- **风险**：
  - 储能系统中标价格持续下降，盈利压力加大
  - 大储强制配储政策执行力度不及预期
  - 海外户储市场去库存周期

## 传导逻辑

```
AI数据中心电力需求 + 新型电力系统建设 → 储能刚需凸显 → 储能装机量爆发 → 龙头市占率提升
```

**当前状态：逻辑链部分确认——储能刚需逻辑清晰，但价格下行压力和盈利模式待改善是主要矛盾。**
