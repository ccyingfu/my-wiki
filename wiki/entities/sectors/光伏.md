# 光伏产业链

## 一句话定位
光伏产业链处于深度产能出清周期，全链开工率<40%，Q1组件招标同比降70%，行业底部尚未确认，产能出清拐点预期在2026年Q3末。

## 产业链结构

```
上游：硅料 → 中游：硅片/电池片 → 下游：组件 → 终端：电站/出口
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-硅料 | 多晶硅致密料 | 过剩率>160%，价格跌至3.67万/吨 |
| 中游-硅片 | N型G10L硅片 | 均价~1.08元/片，同比-36% |
| 中游-电池片 | TOPCon电池片 | 库存8-10天，开始实质性减产 |
| 下游-组件 | TOPCon/BC组件 | 均价0.64-0.68元/W，出口承压 |

## 近期催化 / 风险

- **催化**：十五五规划光伏装机目标调整可能提前产能出清
- **风险**：
  - 全链价格持续阴跌，成本塌陷与需求萎靡形成恶性循环
  - 组件出口退税归零后持续承压
  - 硅料极端报价已至32元/kg，价格战加剧
  - 产能出清拐点预期Q3末，短期不建议抄底

## 传导逻辑

```
硅料价格暴跌 → 硅片/电池片被迫减产(开工率<40%) → 电池库存横盘8-10天(未有效去化) → 组件招标暴跌70%(需求端崩塌) → 装机量偏弱
```

**当前状态：逻辑链断裂——供给端减产已启动但需求端毫无起色，产能出清拐点预期Q3末。**
