# 光通信产业链

## 一句话定位
光通信是AI算力基础设施的核心传输环节，800G/1.6T光模块量价齐升，光芯片是产业链唯一的供给瓶颈。

## 产业链结构

```
上游：光芯片/电芯片 → 中游：光器件/光模块 → 下游：数据中心/电信网络
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-光芯片 | EML/VCSEL/硅光芯片 | 供给缺口>25%，瓶颈环节 |
| 上游-电芯片 | DSP/TIA/Driver | 高速芯片依赖进口 |
| 中游-光器件 | TOSA/ROSA/光组件 | 跟随光模块放量 |
| 中游-光模块 | 800G/1.6T光模块 | 量价齐升，毛利率高位 |
| 下游-数据中心 | 云厂商AI集群 | CAPEX 6150-6500亿$ |
| 下游-电信 | 5G/城域网 | 需求稳健 |
