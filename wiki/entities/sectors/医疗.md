# 医疗产业链

## 一句话定位
医疗板块涵盖设备、创新药、医疗服务和CXO，AI+医疗深度融合带来效率革命，但医药反腐和海外政策风险仍是主要扰动。

## 产业链结构

```
上游：原材料/设备零部件 → 中游：医药制造/医疗服务/CXO → 下游：医院/患者
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-原料 | 化学原料/生物材料 | 供应链稳定 |
| 中游-设备 | 医疗设备/器械 | 国产替代加速，AI辅助诊断渗透 |
| 中游-创新药 | 创新药/生物药 | 国产创新药加速出海 |
| 中游-CXO | CRO/CDMO | 海外政策扰动，国内需求分化 |
| 下游-服务 | 眼科/牙科/体检 | 消费医疗复苏中 |

## 近期催化 / 风险

- **催化**：
  - AI+医疗深度融合，辅助诊断/药物研发效率提升
  - 医疗设备国产替代加速
  - 创新药出海节奏加快
- **风险**：
  - 海外《生物安全法案》等政策影响CXO订单
  - 医药反腐影响行业短期增速
  - 医保控费压力持续

## 传导逻辑

```
AI技术赋能 → 医疗设备智能化/药物研发加速 → 国产替代+出海双驱动 → 龙头市占率提升
```

**当前状态：逻辑链部分确认——AI赋能方向明确，但海外政策风险(CXO)和医保控费压力构成主要矛盾。**
