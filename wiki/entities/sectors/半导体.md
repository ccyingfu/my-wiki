# 半导体产业链

## 一句话定位
半导体是AI时代的基础设施，国产替代+AI算力需求双轮驱动，从设备、材料到晶圆代工全链加速自主化。

## 产业链结构

```
上游：半导体设备/材料 → 中游：晶圆制造(代工) → 下游：封装测试/终端应用
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-设备 | 刻蚀/薄膜/清洗设备 | 国产化率快速提升，订单高景气 |
| 上游-材料 | 硅片/光刻胶/特气 | 依赖进口，国产替代加速 |
| 中游-晶圆代工 | 成熟制程/先进制程 | 成熟制程产能扩张，先进制程攻关中 |
| 下游-封测 | 先进封装/传统封装 | 先进封装(Chiplet/HBM)需求爆发 |
| 终端 | 服务器/手机/汽车/AIoT | AI服务器需求最强，消费电子复苏 |

## 近期催化 / 风险

- **催化**：
  - AI算力需求爆发驱动芯片制造扩产
  - 国产替代政策持续加码，设备/材料自主化加速
  - 成熟制程扩产带来设备订单增量
- **风险**：
  - 美国对华半导体出口管制进一步升级
  - 消费电子复苏力度不及预期
  - 先进制程与国际领先水平差距仍大

## 传导逻辑

```
AI算力需求爆发 + 国产替代政策 → 设备厂商订单高增 → 晶圆厂扩产 → 封测需求提升 → 终端芯片国产化率提升
```

**当前状态：逻辑链部分确认——AI需求和国产替代双驱动，但先进制程仍受限，成熟制程为当前主要增长点。**
