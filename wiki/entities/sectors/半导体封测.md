# 半导体封测产业链

## 一句话定位
半导体封测是AI算力芯片落地的关键环节，先进封装(TSV/CoWoS/Chiplet)产能极度紧缺，成为AI产业链的"产能瓶颈"。

## 产业链结构

```
上游：封装材料/设备 → 中游：封装测试(OSAT) → 下游：芯片设计/Fabless
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-材料 | 封装基板/塑封料/键合线 | 部分依赖进口 |
| 上游-设备 | 键合/测试/切割设备 | 国产替代中 |
| 中游-传统封装 | Wire Bonding/传统FC | 竞争激烈，利润率偏低 |
| 中游-先进封装 | TSV/CoWoS/Chiplet/HBM | 产能紧缺，量价齐升 |
| 下游-设计 | AI芯片/存储芯片/GPU | 需求爆发 |

## 近期催化 / 风险

- **催化**：
  - AI算力芯片(HBM/GPU)需求爆发，先进封装产能紧缺
  - CoWoS等先进封装技术迭代带来价值量提升
  - 国产替代加速，大陆封测厂份额提升
- **风险**：
  - 先进封装技术壁垒高，投资强度大
  - CoWoS产能扩张节奏不确定
  - 传统封装竞争加剧，利润率承压

## 传导逻辑

```
AI算力需求 → HBM/GPU芯片出货增长 → 先进封装(TSV/CoWoS)产能紧缺 → 封测厂商量价齐升
```

**当前状态：逻辑链完整确认——先进封装是AI产业链确定性瓶颈环节，景气度极高。**
