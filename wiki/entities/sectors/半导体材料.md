# 半导体材料产业链

## 一句话定位
半导体材料是芯片制造的"粮食"，AI驱动的HBM需求爆发带来先进封装材料新机遇，国产替代空间巨大(目前自给率<20%)。

## 产业链结构

```
上游：基础化学品/矿产 → 中游：电子特气/光刻胶/封装材料/硅微粉 → 下游：晶圆制造/先进封装
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-基础原料 | 高纯化学品/硅粉 | 依赖进口，国产化起步 |
| 中游-封装材料 | 球硅/Low-α球硅/EMC/Underfill | HBM驱动需求爆发 |
| 中游-电子特气 | 电子级特种气体 | 国产替代加速 |
| 中游-光刻胶 | KrF/ArF光刻胶 | 国产化率极低 |
| 下游-先进封装 | HBM/Chiplet/CoWoS | AI算力需求核心拉动 |

## 近期催化 / 风险

- **催化**：
  - HBM需求爆发驱动先进封装材料放量
  - 国产替代政策支持，半导体材料自主化加速
  - AI算力持续扩张带来上游材料增量需求
- **风险**：
  - 国产材料认证周期长，导入进展不确定
  - 下游晶圆厂资本开支波动影响需求
  - HBM产业链技术迭代快，跟随风险

## 传导逻辑

```
AI算力扩张 → HBM需求爆发 → 先进封装材料需求激增 → 国产材料企业导入放量 → 市占率提升
```

**当前状态：逻辑链部分确认——HBM材料需求方向明确，但国产替代导入节奏需跟踪验证。**
