# 基建产业链

## 一句话定位
基建板块以水泥为核心，当前处于行业低谷期，房地产竣工疲弱拖累需求，但稳增长政策加码和"十五五"规划有望带来需求修复。

## 产业链结构

```
上游：石灰石/煤炭 → 中游：水泥/混凝土/钢材 → 下游：房地产/基建/市政工程
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-原料 | 石灰石/煤/电力 | 成本相对稳定 |
| 中游-水泥 | 熟料/水泥 | 价格低迷，行业亏损面扩大 |
| 中游-混凝土 | 商品混凝土 | 需求偏弱 |
| 中游-钢材 | 螺纹钢/线材 | 价格震荡 |
| 下游-房地产 | 房屋竣工 | 持续疲弱，拖累需求 |
| 下游-基建 | 交通/水利/市政 | 稳增长发力方向 |

## 近期催化 / 风险

- **催化**：
  - 稳增长政策加码，专项债发行加速
  - "十五五"基建规划带来中长期需求
  - 水泥行业协同限产改善供需格局
- **风险**：
  - 房地产竣工持续疲弱，核心需求拖累
  - 水泥行业产能过剩，价格战加剧
  - 基建投资增速不及预期

## 传导逻辑

```
稳增长政策 → 基建投资加码 → 水泥/钢材需求改善 → 行业供需格局优化 → 龙头盈利修复
```

**当前状态：逻辑链待验证——行业处于低谷期，需等待房地产端企稳和基建发力信号。**
