# 存储芯片产业链

## 一句话定位
存储芯片进入AI驱动的"超级周期"，Q1 DRAM合约价暴涨90-95%，Q2预测续涨58-63%，原厂弃短约转3-5年长约，定价权完全在卖方。

## 产业链结构

```
上游：晶圆/存储原厂 → 中游：存储颗粒/模组 → 下游：终端应用
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-原厂 | DRAM/NAND晶圆 | 稼动率~90%+，利润突破历史峰值 |
| 中游-模组 | 内存条/SSD模组 | 利润弹性释放 |
| 下游-终端 | 服务器/PC/手机 | AI服务器需求爆发，终端成本承压 |

## 近期催化 / 风险

- **催化**：
  - 三大原厂转向3-5年长约，定价权在卖方
  - AI服务器需求爆发驱动
  - 原厂产能转向HBM/Server，常规产品供给不足
- **风险**：
  - Q2涨幅较Q1收窄（63% vs 95%），市场可能解读为涨价动能减弱
  - 终端成本上扬可能抑制需求
  - DRAM利润率突破历史峰值=周期顶部特征，需关注拐点信号

## 传导逻辑

```
原厂产能转向HBM/Server → 常规DRAM/NAND供给不足 → Q1合约价暴涨90-95% → 模组厂利润弹性释放 → 终端被动接受涨价(3-5年长约)
```

**当前状态：逻辑链完整确认——AI驱动的"超级周期"，定价权完全在卖方。**
