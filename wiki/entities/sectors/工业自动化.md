# 工业自动化产业链

## 一句话定位
工业自动化是制造业升级的核心，从工控核心部件(伺服/变频/PLC)到新能源车电控系统，国产替代+产业升级双轮驱动。

## 产业链结构

```
上游：芯片/元器件 → 中游：工控产品(伺服/变频/PLC) → 下游：OEM设备/终端制造业
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-元器件 | IGBT/MCU/传感器 | 部分依赖进口 |
| 中游-伺服系统 | 伺服电机/驱动器 | 国产替代加速 |
| 中游-变频器 | 低压/中压变频器 | 龙头份额提升 |
| 中游-PLC | 可编程逻辑控制器 | 国产化率仍低，突破中 |
| 中游-电控系统 | 新能源车电控 | 乘用车市场放量 |
| 下游-OEM | 3C/锂电/光伏设备 | 景气分化 |

## 近期催化 / 风险

- **催化**：
  - 新能源车电控业务放量
  - 工控国产替代加速，份额持续提升
  - 制造业升级/机器人等新方向
- **风险**：
  - 传统制造业投资放缓
  - 新能源车电控竞争加剧
  - 宏观经济不确定性影响资本开支

## 传导逻辑

```
制造业升级 + 国产替代 → 工控产品需求增长 → 龙头份额提升 + 新能源车电控放量 → 业绩双驱动
```

**当前状态：逻辑链部分确认——国产替代趋势明确，但短期受制造业景气波动影响。**
