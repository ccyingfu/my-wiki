# 新能源产业链

## 一句话定位
新能源以动力电池为核心，全球电动化+储能需求双驱动，但行业面临产能结构性过剩，龙头企业凭借技术和成本优势加速出清。

## 产业链结构

```
上游：锂/镍/钴资源 → 中游：正负极/隔膜/电解液/电池 → 下游：新能源车/储能系统
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-资源 | 碳酸锂/镍钴 | 价格高位回落，供给扰动减弱 |
| 中游-材料 | 正极/负极/隔膜/电解液 | 产能过剩，价格持续下行 |
| 中游-电池 | 磷酸铁锂/三元电池 | 龙头市占率提升，产能利用率52% |
| 下游-储能 | 储能系统/集成 | 储能需求爆发，成为第二增长曲线 |
| 下游-整车 | 新能源乘用车 | 渗透率52.9%，总量承压 |

## 近期催化 / 风险

- **催化**：
  - 储能需求爆发，AI数据中心配储需求新增量
  - 龙头成本优势持续扩大，市占率加速提升
  - 以旧换新政策落地带动新能源车需求
- **风险**：
  - 电池产能利用率仅52%，行业仍处结构性过剩
  - 碳酸锂价格波动影响产业链利润分配
  - 海外贸易摩擦(关税/政策壁垒)影响出口

## 传导逻辑

```
锂资源价格回落 → 中游成本改善 → 电池装机高增 → 新能源车渗透率提升 + 储能爆发 → 龙头市占率提升
```

**当前状态：逻辑链部分确认——中游装机数据良好，但产能过剩和价格战是主要矛盾。**
