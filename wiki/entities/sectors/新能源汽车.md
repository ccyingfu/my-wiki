# 新能源汽车产业链

## 一句话定位
新能源汽车渗透率突破52.9%创历史新高，但整体车市零售同比下降15%，总量萎缩背景下渗透率提升"含金量"存疑，等待Q2以旧换新政策落地验证。

## 产业链结构

```
上游：锂资源 → 中游：动力电池 → 下游：整车
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-锂资源 | 碳酸锂 | 16-16.5万/吨，从19万回落 |
| 中游-动力电池 | 磷酸铁锂/三元 | 装机+35%，产能利用率52% |
| 下游-整车 | 新能源乘用车 | 渗透率52.9%，总量-15% |

## 近期催化 / 风险

- **催化**：
  - Q2以旧换新政策落地细则（总量修复关键）
  - Q2传统旺季
- **风险**：
  - 整体车市零售-15%，总量萎缩
  - 碳酸锂高位回落，津巴布韦配额政策执行力度存疑
  - 电池产能利用率仅52%，行业仍处结构性过剩

## 传导逻辑

```
碳酸锂从19万回落至16万(供给扰动减弱) → 电池装机高增(+30-40%) → 整车渗透率破52.9%(历史拐点) 但总量下降15%
```

**当前状态：逻辑链部分确认——中游装机和下游渗透率数据良好，但总量萎缩和碳酸锂高位回落形成矛盾信号。**
