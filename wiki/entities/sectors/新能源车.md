# 新能源车产业链

## 一句话定位
新能源车渗透率突破52.9%创历史新高，行业从电动化竞争转向智能化下半场，龙头凭借垂直整合优势持续扩大领先。

## 产业链结构

```
上游：锂/钴/镍资源 → 中游：三电系统(电池/电机/电控) + 智能化(芯片/传感器) → 下游：整车制造/销售
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-资源 | 碳酸锂 | 16-16.5万/吨，震荡 |
| 中游-电池 | 动力电池 | 装机+35%，产能利用率52% |
| 中游-电控 | 电机/电控 | 国产化率高 |
| 中游-智能化 | 自动驾驶/智能座舱 | 智能化成为竞争焦点 |
| 下游-整车 | 新能源乘用车 | 渗透率52.9%，总量-15% |

## 近期催化 / 风险

- **催化**：
  - Q2以旧换新政策落地，总量修复
  - 智能驾驶技术突破(城市NOA普及)
  - 出海加速(欧洲/东南亚市场)
  - 龙头垂直整合成本优势持续扩大
- **风险**：
  - 整体车市零售-15%，总量萎缩
  - 价格战加剧，行业盈利承压
  - 海外关税壁垒影响出海节奏

## 传导逻辑

```
以旧换新政策 + 智能化升级 → 终端需求修复 → 渗透率持续提升 → 龙头市占率扩大
```

**当前状态：逻辑链部分确认——渗透率数据良好，但总量萎缩和价格战是主要矛盾，等待Q2政策验证。**
