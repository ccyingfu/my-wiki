# 有色金属产业链

## 一句话定位
铜价$12,920创历史新高，矿端极度紧缺（TC零或负值）推动价格上行，但冶炼端利润为负被迫减产，利润分配极端——矿山赚满而冶炼亏损。

## 产业链结构

```
上游：矿山/LME定价 → 中游：冶炼/加工 → 下游：终端需求
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-矿山 | 铜精矿 | 矿端紧缺，TC零或负值 |
| 中游-冶炼 | 精炼铜/铝 | 利润为负，被迫减产 |
| 下游-需求 | 电网/新能源/建筑 | 电网+新能源驱动，房地产拖累 |

## 近期催化 / 风险

- **催化**：
  - 矿端紧缺→冶炼减产→库存去化→价格上行逻辑链完整
  - 电网投资+新能源车铜需求双轮驱动
  - 冶炼大规模减产可能带来供给超预期收缩
- **风险**：
  - 冶炼利润严重负值为不可持续状态
  - 房地产竣工疲弱拖累建筑用铜/铝需求
  - TC负值倒逼减产，但减产效果需时间验证

## 传导逻辑

```
矿端紧缺(TC零或负值) → 冶炼利润为负被迫减产 → 全球库存持续去化 → 铜价$12,920(+29%) → 但房地产竣工拖累终端需求
```

**当前状态：逻辑链部分确认——矿端紧缺→价格上行逻辑已验证，但冶炼端利润为负属不可持续状态。**
