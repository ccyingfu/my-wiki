# 消费产业链

## 一句话定位
消费板块以白酒+中药为核心，品牌护城河深厚，ROE极高且稳定，是AI无法替代的物理防御性资产。

## 产业链结构

```
上游：原材料(粮食/药材) → 中游：生产酿造/制药 → 下游：渠道经销 → 终端：消费者
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-原料 | 高粱/小麦/中药材 | 价格相对稳定 |
| 中游-白酒 | 高端/次高端白酒 | 行业去库存进行中，龙头韧性足 |
| 中游-中药 | 中成药/名贵中药 | 品牌中药提价逻辑 |
| 下游-渠道 | 经销商/直销/电商 | 渠道改革深化 |

## 近期催化 / 风险

- **催化**：
  - 消费刺激政策加码(以旧换新扩围)
  - 白酒去库存接近尾声，渠道预期改善
  - 中药提价+医保目录调整受益
- **风险**：
  - 白酒行业去库存压力仍存
  - 消费降级趋势延续，高端消费承压
  - 房地产财富效应消退影响消费意愿

## 传导逻辑

```
消费刺激政策 + 去库存尾声 → 白酒终端动销改善 → 龙头量价企稳 → ROE维持高位 → 估值修复
```

**当前状态：逻辑链待验证——龙头业绩韧性足，但行业整体仍处去库存周期，需等待终端动销拐点信号。**
