# 物流产业链

## 一句话定位
物流板块以快递和供应链为核心，AI赋能降本增效，行业从价格战转向价值竞争，龙头企业凭借网络效应和科技投入构筑壁垒。

## 产业链结构

```
上游：运输工具/仓储设施 → 中游：快递/快运/供应链服务 → 下游：电商/制造业/消费者
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-基础设施 | 航空/陆运/仓储 | 网络布局完善 |
| 中游-快递 | 时效件/经济件 | 行业格局稳定，龙头溢价 |
| 中游-供应链 | 一体化供应链方案 | 增长最快赛道 |
| 中游-国际物流 | 跨境物流/货代 | 出海物流需求增长 |
| 下游-电商 | 电商件/直播电商 | 消费需求分化 |
| 赋能-AI物流 | 智能分拣/路径优化 | 降本增效 |

## 近期催化 / 风险

- **催化**：
  - AI赋能物流(智能分拣/路径优化)持续降本
  - 供应链业务成为第二增长曲线
  - 直播电商/跨境电商带动物流需求
  - 行业价格战趋缓，龙头份额提升
- **风险**：
  - 消费疲弱影响快递业务量
  - 航空燃油成本波动
  - 价格战反复风险

## 传导逻辑

```
消费复苏 + 电商增长 → 快递业务量增长 → AI降本增效 → 龙头盈利能力改善
```

**当前状态：逻辑链部分确认——龙头壁垒稳固，但短期受消费景气波动影响，需关注价格战演变。**
