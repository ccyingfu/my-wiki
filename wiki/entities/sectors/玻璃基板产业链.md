# 玻璃基板产业链

## 一句话定位
AI芯片先进封装的核心基材，2026年商业化元年，全球88%被美日三家企业垄断，国产替代空间巨大。

## 产业链结构

### 概念定义
玻璃基板（Glass Substrate）是以特种玻璃为基材的薄板载体，主要功能为TFT、半导体芯片等提供平坦、稳定的支撑平台。核心优势：强度可调节、耐高温、表面超光滑（比有机材料光滑5000倍）。

### 分类
| 分类 | 用途 | 市场阶段 |
|------|------|---------|
| TFT-LCD玻璃基板 | 液晶显示面板核心组件 | 成熟市场 |
| OLED玻璃基板 | 高端手机/电视显示面板 | 成熟市场 |
| **半导体封装玻璃基板** 🔥 | AI芯片、HBM先进封装 | 2026年小批量商业化 |
| 光掩模基板 | 芯片光刻工艺掩模版 | 成熟市场 |
| 盖板玻璃 | 手机/手表屏幕保护 | 成熟市场 |

### 技术驱动（为什么突然火了）
- AI芯片尺寸越来越大（Nvidia B200 GPU超8000mm²），传统有机基板高温翘曲变形
- 玻璃基板优势：连接密度提升10倍、降低50%功耗、面板级封装大幅提升利用率
- **巨头布局**：台积电CoPoS 2026年6月建成试点线；英特尔10亿美元+产线已大规模量产；三星电机向苹果送样；苹果"Baltra"AI芯片已测试

### 产业链全景
```
上游（原材料/设备）         中游（玻璃基板制造）         下游（应用终端）
石英砂/硼酸/氧化铝  ─────→  玻璃基板生产  ─────→  显示面板(京东方/TCL)
激光切割/镀膜设备          熔炼→成型→精加工→检测       先进封装(台积电/英特尔)
精密检测设备                                        PCB/IC载板

竞争格局：康宁(48%) + AGC(23%) + 电气硝子(17%) = 88%份额
```
