# 电力设备产业链

## 一句话定位
电力设备是新型电力系统和AI数据中心的基础设施，电网投资持续加码，智能化+数字化升级带来增量需求。

## 产业链结构

```
上游：铜/铝/元器件 → 中游：一次设备(变压器/开关) + 二次设备(继保/自动化) → 下游：电网/工商业
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-原材料 | 铜/铝/硅钢 | 铜价高位，成本端压力 |
| 中游-一次设备 | 变压器/GIS/开关柜 | 需求旺盛，产能偏紧 |
| 中游-二次设备 | 继电保护/自动化系统 | 智能化升级驱动 |
| 中游-数字化 | 电网数字化/调度系统 | AI赋能新方向 |
| 下游-电网 | 国家电网/南方电网 | 投资持续加码 |
| 下游-数据中心 | IDC供电/配储 | AI新增量 |

## 近期催化 / 风险

- **催化**：
  - 电网投资持续加码，特高压+配电网建设
  - AI赋能电网智能化(调度/巡检/运维)
  - 新型电力系统建设带来设备升级需求
  - 数据中心供电需求新增量
- **风险**：
  - 铜价高位压缩设备厂商利润
  - 电网投资节奏不确定性
  - 竞争加剧

## 传导逻辑

```
新型电力系统 + AI数据中心 → 电网投资加码 + 智能化升级 → 电力设备需求增长 → 龙头订单放量
```

**当前状态：逻辑链部分确认——电网投资方向明确，AI赋能是新增长点，但需关注铜价成本和投资节奏。**
