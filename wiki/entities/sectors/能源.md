# 能源产业链

## 一句话定位
清洁能源(水电+核电)是AI数据中心电力需求的长期受益者，现金流极优，具备类债券属性的高股息防御性板块。

## 产业链结构

```
上游：水力/核能资源 → 中游：发电运营 → 下游：电网输送/电力交易
```

| 环节 | 关键产品 | 当前状态 |
|------|---------|---------|
| 上游-资源 | 水电装机/核电机组 | 稳定增长，新项目审批加速 |
| 中游-发电 | 水力发电/核电 | 利用率稳定，成本优势明显 |
| 下游-电网 | 电力输送/交易 | AI数据中心用电需求激增 |

## 近期催化 / 风险

- **催化**：
  - AI数据中心电力需求激增，长期合同锁定
  - 核电审批加速，"十五五"清洁能源装机目标上调
  - 高股息+稳定现金流，防御属性突出
- **风险**：
  - 来水偏枯影响水电出力
  - 电价市场化改革带来价格波动
  - 核安全事故风险(低概率高影响)

## 传导逻辑

```
AI数据中心用电激增 → 电力需求结构性增长 → 清洁能源(水电/核电)受益 → 现金流稳定+高股息 → 估值修复
```

**当前状态：逻辑链部分确认——AI电力需求长期逻辑清晰，但短期催化依赖来水情况和政策节奏。**
